This is a modified version ofSPACE created by Nick Sampson. 

Modifications:
Assessment now how a roughly fixed time cost.
Reverse tandem runs have been added.
Reverse tandem runs no longer include ants which have just been dropped.
The number of initial scouts is a function of quorum threshold.
Recruiters only try to recruit for a set amount of time before giving up.
Assessment now has a truly fixed cost.
